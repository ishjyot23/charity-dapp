Title: CharityChain DApp – Blockchain-Based Transparent Donation System

Description:
Developed a decentralized charity donation platform using Ethereum smart contracts to address lack of transparency, fraud, and poor accountability in traditional donation systems. The DApp implements NGO registration and cause creation via a CharityRegistry contract, secure ETH donations and fund tracking through a DonationVault contract, and immutable auditing of all transactions using an AuditLogger contract. Built using Solidity, Ganache, Truffle Suite, and MetaMask, the system ensures every donation is traceable on-chain, with verifiable NGO causes and automated, trustless fund management.
